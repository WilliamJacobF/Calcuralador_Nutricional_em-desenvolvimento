# Projeto-Calculadora-Nutricional
 Projeto de extenção do curso de Engenharia de Software, Um site que mostra as Calorias, Proteínas, Carboidratos e Gorduras dos Alimentos buscados, feito
 em Laravel usando php, HTML, CSS, Bootstrap, ligado a um banco de dados relacional(mysql).

---

## ❓ O que é um Projeto de Extenção
- disciplina acadêmica que permite uma atuação junto à comunidade por meio da implementação de um projeto com foco em soluções coletivas e que traga melhorias para um grupo de pessoas daquela localidade.

---

## 📌 Funcionalidades

- Busca por nome do alimento e quantidade requisitada
- Exibe as Calorias, Proteínas, Carboidratos e Gorduras dos alimentos pela quantidade em gramas

---

## ⚙️ Tecnologias Utilizadas

### Backend:
- Laravel 12
- PHP 8.1
- MySQL

### Frontend:
- HTML5
- CSS3
- JavaScript
- Bootstrap

---

## 🚀 Como Executar

### Backend (Laravel):

1. Clone o repositório:
```bash
git clone https://github.com/WilliamJacobF/Projeto-Calculadora-Nutricional.git
```

2. Instale as dependências:
```bash
composer install
```

3. Configure o arquivo `.env` com os dados do banco de dados.

4. Rode as migrações:
```bash
php artisan migrate
```

5. Inicie o servidor:
```bash
php artisan serve
```

### Frontend:

1. Acesse a pasta `views/`
2. Abra o arquivo index.blade.php no navegador.
