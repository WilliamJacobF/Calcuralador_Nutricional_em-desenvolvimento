# Projeto-Calculadora-Nutricional
 Este é um sistema fullstack desenvolvido com **Laravel** no backend e **HTML/CSS/JavaScript** no frontend. Ele permite aos moradores consultar os **dias e horários da coleta de lixo** digitando o nome da rua em um formulário.

---

## 📌 Funcionalidades

- Busca por nome do alimento e quantidade requisitada
- Exibe as Calorias, Proteínas, Carboidratos e Gorduras dos alimentos pela quantidade em gramas

---

## ⚙️ Tecnologias Utilizadas

### Backend:
- Laravel 12
- PHP 8.1
- MySQL

### Frontend:
- HTML5
- CSS3
- JavaScript
- Bootstrap

---

## 🚀 Como Executar

### Backend (Laravel):

1. Clone o repositório:
```bash
git clone https://github.com/WilliamJacobF/Projeto-Calculadora-Nutricional.git
```

2. Instale as dependências:
```bash
composer install
```

3. Configure o arquivo `.env` com os dados do banco de dados.

4. Rode as migrações:
```bash
php artisan migrate
```

5. Inicie o servidor:
```bash
php artisan serve
```

### Frontend:

1. Acesse a pasta `views/`
2. Abra o arquivo index.blade.php no navegador.
